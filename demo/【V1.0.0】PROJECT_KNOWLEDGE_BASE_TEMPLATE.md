### 4.2 功能详情

#### F001：成本预估次数显示逻辑

**功能描述：** 优化成本预估次数(ce_count)字段的显示逻辑，当该字段没有数值时显示为'-'，提升数据展示的一致性和可读性

**入口路径：** 品牌工作台 → 列表页面

**功能逻辑：**

1. 系统加载包含ce_count字段的页面或报表
2. 前端检查ce_count字段是否有数值
3. 如果有数值，正常显示该数值
4. 如果没有数值（null、undefined或空值），显示为'-'
5. 当ce_count字段为0时，正常显示数字0，不显示为'-'

**特殊逻辑说明：**

- 当ce_count字段为null、undefined或空值时，显示为'-'
- 当ce_count字段为0时，正常显示数字0

**权限控制：**

| 角色 | 操作权限 | 数据范围 |
|------|----------|----------|
| 超级管理员 | 查看 | 全量数据 |
| 普通用户 | 查看 | 授权范围 |
| 只读角色 | 查看 | 授权范围 |
